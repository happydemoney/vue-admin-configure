# vue-admin-configure

vue admin template base on typescript

## Getting started

```
# clone code to local 
git clone https://github.com/happydemoney/vue-admin-configure.git

# enter the project directory
cd vue-admin-configure

# install dependency
yarn install

# develop
yarn serve
```
