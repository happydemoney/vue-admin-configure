# vue-admin-ts

vue admin template base on typescript

## Getting started

```
# clone code to local 
git clone https://github.com/happydemoney/vue-admin-ts.git

# enter the project directory
cd vue-admin-ts

# install dependency
yarn install

# develop
yarn serve
```
